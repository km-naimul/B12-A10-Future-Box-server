# FinEase – Personal Finance Management App  
Live Site: https://extraordinary-paprenjak-cd6d76.netlify.app/

---

##  Features

- **Smart Expense Tracking** – Add, edit, delete & categorize your expenses easily.
- **Visual Reports & Charts** – Beautiful graphs to track income vs expenses.
- **Monthly Budget System** – Set budgets and see real-time usage progress.
- **Savings Goal Tracker** – Create savings goals with animated progress UI.
- **Secure Authentication** – Login/Register with Firebase Authentication.
- **Responsive UI** – Fully optimized for mobile, tablet, and desktop devices.
- **Dark/Light Theme** – Switch between modern color themes instantly.
- **User Dashboard** – View all transactions, budgets, and analytics in one place.

---

## Technologies Used

- **React.js**
- **Tailwind CSS**
- **DaisyUI**
- **Firebase Auth**
- **Recharts**
- **Node.js + Express (Backend)**
- **MongoDB (Database)**
